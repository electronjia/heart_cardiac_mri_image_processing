This "Sunnybrook Cardiac MR Database" is made available under the CC0 1.0 Universal license as described in the CC0_License.htm html page in this directory, with a more detail here:
http://creativecommons.org/publicdomain/zero/1.0/

The Sunnybrook Cardiac MR Database includes both DICOM studies and associated contours, and these data are available under this license.

Please read the ODC_Community_norms.pdf in this directory or a more current version here:
http://www.opendatacommons.org/norms
The Community Norms document reflects norms of access and use that can help create a vibrant community around data and databases that would be great if you would voluntarily adopt.
